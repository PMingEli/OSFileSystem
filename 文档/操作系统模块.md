# 功能

**多用户 ：**usr1, …, usr8 (1-8个用户)

**多级目录：**可有多级子目录

**用户登录：**login ()

**系统初始化：**建文件卷、提供登录模块

**文件的创建：** create

**文件的打开：**open

**文件的读：**read

**文件的写：**write

**文件关闭：**close

**删除文件：**delete

**创建目录：**mkdir

**改变目录：**chdir

**列出文件目录：**dir

**退出：**logout

**格式化：**format



### 用户：创建，登录，退出

### 文件：创建，删除，打开，关闭，读写

### 目录：创建，改变，多级目录，列出文件目录
